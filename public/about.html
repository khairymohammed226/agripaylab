About This Project

This Online Banking System is a full-stack web application designed to simulate real banking operations in a secure and modern way.

Main Features

  • User Registration & Secure Login
  • ATM Withdraw & Deposit Operations
  • Bank Transfers
  • Card Management
  • Transaction History

Technologies Used

  • HTML, CSS, JavaScript
  • Node.js & Express
  • MongoDB & Mongoose
  • JWT Authentication

Purpose

The project was created for educational purposes to demonstrate how modern online banking systems work, including authentication, transactions, and database integration.
