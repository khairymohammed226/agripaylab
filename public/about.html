About this project

This Online Banking System is a full-stack web application designed to simulate real banking operations in a secure and modern way. Built as an educational project to demonstrate authentication flows, database integration, and financial transaction logic — covering everything from account creation to secure fund transfers.

Main features

User registration & secure login
ATM withdraw & deposit operations
Bank transfers between accounts
Card management system
Full transaction history
Real-time balance updates

Project timeline

Week 1
Project planning & setup
Defined architecture, assigned roles, set up Node.js + MongoDB environment and Git workflow.
Week 2–3
Authentication & user system
Built registration, login, and JWT token management with role-based access control.
Week 4–5
Core banking features

Week 6
UI polish & testing
Final design pass, end-to-end testing, bug fixes, and documentation.

Purpose

This project was created for educational purposes to demonstrate how modern online banking systems work end-to-end — from secure user authentication and session management, to financial transactions and database integration. It serves as a hands-on learning experience for full-stack development best practices.
